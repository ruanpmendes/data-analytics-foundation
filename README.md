# Fundação em Data Analytics & Engineering 🧱

Bem-vindo(a) ao meu repositório de estudo e prática diária. Este espaço é dedicado a documentar minha evolução contínua em Análise de Dados, Engenharia de Dados e Business Intelligence.

## 🎯 Objetivo
Construir uma base sólida em manipulação de dados, regras de negócio e otimização de performance através da prática diária. Este repositório serve como um diário de bordo da minha jornada técnica.

## 🗂️ Estrutura do Repositório

* **`01_Excel_PowerQuery/`**: Modelagem avançada de dados, despivotamento (unpivot) e scripts em linguagem M.
* **`02_PowerBI_DAX/`**: Modelagem Star Schema, inteligência de tempo e medidas DAX avançadas.
* **`03_SQL/`**: Consultas complexas, Joins, CTEs e Window Functions.
* **`04_Python_Pandas/`**: Operações vetorizadas, scripts ETL e otimização de memória.
* **`05_Business_Metrics/`**: KPIs executivos (LTV, CAC, Churn) e resoluções de SQL do HackerRank/LeetCode.
