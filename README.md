# Data Analytics & Engineering Foundation 🧱

Welcome to my daily study and practice repository. This space is dedicated to tracking my continuous evolution in Data Analytics, Data Engineering, and Business Intelligence.

## 🎯 Objective
To build a rock-solid foundation in data manipulation, business rules, and performance optimization through daily practice. This repository serves as a logbook of my technical journey.

## 🗂️ Repository Structure

* **`01_Excel_PowerQuery/`**: Advanced data modeling, unpivoting, and M language scripts.
* **`02_PowerBI_DAX/`**: Star schema modeling, time intelligence, and advanced DAX measures.
* **`03_SQL/`**: Complex queries, Joins, CTEs, and Window Functions.
* **`04_Python_Pandas/`**: Vectorized operations, ETL scripts, and memory optimization.
* **`05_Business_Metrics/`**: Executive KPIs (LTV, CAC, Churn) and HackerRank/LeetCode SQL solutions.
